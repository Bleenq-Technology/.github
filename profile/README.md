# Bleenq

**Bleenq LLC** is a boutique **AI/ML consulting firm and incubator**. We design and build
production-grade AI/LLM platforms and applications — with the operational maturity, security, and
observability that take systems from impressive demo to dependable production.

## Open source

- **[ai-homelab](https://github.com/Bleenq-Technology/ai-homelab)** — a production-style,
  self-hostable **AI/LLM platform**: ~40 services as modular Docker Compose (identity, secrets, a
  full observability stack, and a GPU-accelerated AI suite) behind Traefik with automatic wildcard
  TLS. An open reference architecture you can study, adapt, and run yourself. *A "homelab" in name
  only.*
